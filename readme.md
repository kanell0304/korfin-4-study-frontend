[git 기본 명령어]
git pull -> 원격 저장소의 데이터를 로컬 저장소로 가져오기 <br/>
git add . -> 수정 사항을 스테이지에 올림 <br/>
git commit -m '커밋 메세지' -> 스테이지에 올려진 사항을 로컬 저장소에 저장 <br/>
git commit -am '커밋 메세지' -> 위 2개의 명령어들을 한번에 수행 <br/>
git push -u origin main -> main 브랜치에 업스트림 푸쉬 (처음 한번 이후 'git push'로 단축 가능) <br/>
