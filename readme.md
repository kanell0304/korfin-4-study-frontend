핀테크 과정 4조 스터디 frontend

[사용 방법]
1. git bash 설치
2. 원하는 로컬 경로에 폴더 생성
3. vscode를 켜고 해당 폴더 열기
4. ctrl + shift + `단축키로 터미널 열기
5. 터미널 '+'키 옆을 클릭 수 git bash 열기
6. git init
7. git clone https://github.com/kanell0304/kofin-4-study-frontend.git .

[git 기본 명령어]
git pull -> 원격 저장소의 데이터를 로컬 저장소로 가져오기
git add . -> 수정 사항을 스테이지에 올림
git commit -m '커밋 메세지' -> 스테이지에 올려진 사항을 로컬 저장소에 저장
git commit -am '커밋 메세지' -> 위 2개의 명령어들을 한번에 수행
git push -u origin main -> main 브랜치에 업스트림 푸쉬 (처음 한번 이후 'git push'로 단축 가능)
